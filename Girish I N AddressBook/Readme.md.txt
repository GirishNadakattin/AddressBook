# Address Book in C

A simple Address Book application written in C that allows users to add, view, search, edit, and delete contacts.

## Features
- Add new contact
- Search contact by name
- Edit existing contact
- Delete contact
- Save contacts using file handling

## Technologies Used
- C language
- File Handling
- Structures

## How to Run
1. Compile the code using GCC:
   ```bash
   gcc main.c add.c edit.c delete.c search.c -o addressbook
